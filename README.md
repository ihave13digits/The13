# The13

A horror game written in Godot

Keyboard Controls:

* W S A D - Move
* Mouse - Pan
* Left Click - Interact
* Right Click - Aim Flashlight
* Esc - Toggle Mouse Capture

Joypad Controls:

* Left Joystick - Move
* Right Joystick - Pan
* R2 - Interact
* L2 - Aim Flashlight
* Start - Toggle Mouse Capture
